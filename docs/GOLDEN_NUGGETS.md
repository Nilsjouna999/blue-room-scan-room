# BLUE ROOM — Golden Nuggets

Layer: CANON · Status: living (core LOCKED · strong = adopted · watch = provisional) · 2026-06-14

The distilled, absolute, generative one-liners that make planning + coding precise. High bar —
the badge is for insights that change the game, not rules or details. Distilled + stress-tested
(23-agent pass). Fire `/gods-finger` on a candidate to test a new one against the bar.

## The bar (a line is golden only if ALL five clear; default KILL)
1. Absolute & core (whole project, not one surface/detail).
2. Generative (you can derive many calls from it).
3. A crisp one-liner.
4. Non-redundant (merge if two overlap).
5. An insight, not a rule ("no 0–100" is a rule; "reads the photo, not the person" is a nugget).

## Core (the spine — LOCKED)
1. **Reads the photo, not the person** — the artifact is the subject; the human never is.
   *Note (2026-06-14): refined — we DO read the IMAGE-ACT (posture/gesture/role, re-authorable),
   never the PERMANENT (worth/identity); the wall holds per-read AND in aggregate. See
   `docs/HUMAN_READ_LINE_V1.md`.*
2. **Blue Room mints the artifact** — develops what the photo already held into a filed, addressable record.
3. **The card is the crown, not the kingdom** — the page/universe is the main dish; the card links back.
4. **The site carries the depth; the card stays compressed.**
5. **Kind, not amount** — Free is the complete front; paid is the *same scan developed deeper* (a mode,
   not a grade). *Reframed 2026-06-14 from "sealed back of the card" — the card links to a deeper record,
   not a literal back.*
6. **Latent→developed is the physics** — sealed reads "not yet developed," never "withheld."
7. **Magnetism is pull by giving, not exploiting** — provenance made legible, never scarcity/streaks.

## Strong (adopted)

9. The photo creates the card; the profile adds the lore — never inferred.
10. The room is a ruled instrument sheet; the card is the only object with depth.
11. The magnetism is already built — surface live state, never add machinery.
12. Every outcome is a win — a low read is a different kind of artifact, never a worse person.
13. The voice is deadpan archival — analyst dead serious, situation absurd.
14. Reframe is camera direction, not self-coaching.
