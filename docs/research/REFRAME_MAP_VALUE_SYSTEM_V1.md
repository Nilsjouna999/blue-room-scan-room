# REFRAME_MAP_VALUE_SYSTEM_V1 — Distilled

## Master principle
The Reframe Map shows what the artifact did — and what a different artifact would do.

It is camera direction for the next pull, not coaching for the self.

## Main test
If the suggested change requires the person to be different, it is banned.
If it only requires the photo to be different, it is allowed.

Bad:
Stand taller to look more confident.

Good:
Lower the crop so the posture reads full-height in frame.

## Product role
Reframe Map = the card’s production notes.

It explains:
- why this card landed
- which visual levers created it
- what different levers would create a different card

A variant is another card, not a fixed/better card.
