## Master law

BLUE ROOM reads the photo, not the person.

More precise:
BLUE ROOM grades the artifact like a card, gem, print, or signal — never appraises the human like a profile.

## Artifact Test

If a different photo of the same person could change the reading, it is an artifact signal.

If the reading sticks to the person regardless of photo, it is banned.

Allowed:
`visible cue → image effect`

Banned:
`visible cue → person truth`

## Human-in-frame law

Read the performance, not the performer.
Read the frame, not the face behind it.

Allowed:
gesture, gaze, expression as tension, posture as silhouette, styling as color/shape, body angle as geometry, subject placement, relationship to setting/objects.

Banned:
attractiveness, confidence/insecurity, dominance/status, personality diagnosis, trust/danger, gender/age/race/health/class inference.

Five-gate test:
1. Subject is photo/frame/card/scene, not “you”.
2. A different photo could change the read.
3. Visible performance, not inner psychology.
4. Image effect, not human worth.
5. No identity inference.

## Reframe Map law

The Reframe Map changes the artifact, not the person.

It shows what the artifact did — and what a different artifact would do.
It is camera direction for the next pull, not coaching for the self.

Test:
If the suggested change requires the person to be different, it is banned.
If it only requires the photo to be different, it is allowed.

Bad:
Stand taller to look more confident.

Good:
Lower the crop so the posture reads full-height in frame.

## Reframe Map product role

Reframe Map = the card’s production notes.

It explains:
- why this card landed
- which visual levers created it
- what different levers would create a different card

A variant is another card, not a fixed/better card.

Never say:
stronger card, better version, fix this, improve yourself, unlock your potential, true self.

Say:
different pull, alternate route, cleaner Signal route, Cabin Context variant.

## Reframe Map categories
