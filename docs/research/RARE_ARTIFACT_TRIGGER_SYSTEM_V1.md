# RARE_ARTIFACT_TRIGGER_SYSTEM_V1 — Distilled

## Master principle
BLUE ROOM rarity is not a prize. It is a catalog note.

Rare means uncommon artifact configuration, not rare person.

## V1 model
Deterministic rule dictionary:

visible cue present → trigger fires → notation/stamp/comment available

No chance. No packs. No rerolls. No fake rarity. No “try again for rare”.
