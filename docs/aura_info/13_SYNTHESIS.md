## 2. The five marriage rulings (where F1 had to weld, not just stack)

**R1 — The two-tier label (resolves O2-ruling-(c) vs O4's Tier-A/Tier-B, and C-14).** The KIND is a **closed backstage enum** (Standoff / Edge / Figure-Ground / Light-Fit / Vector-into-Void / null): it constrains the schema, drives the default residue direction, feeds QA and the aria summary — and **never renders as visible text**. The classLine (authored contradiction pair, e.g. "Warm Refusal") is the **single rendered designation**. This honors BR-S109's singularity law (ONE struck designation, no code-row) AND the Gate's schema constraint AND doc 01's charge. O2's core demand was singularity + geometry-constraint at the schema — both satisfied; O2's "render the geometry noun itself" is refined, not overruled, because a bare "Figure-Ground" chip would read as a second Metrics taxonomy term while the classLine carries the verdict-heat only Aura may carry. Social nouns (refusal, labour, witness) are legal in a classLine ONLY gesture-pinned (RULE 0's one allowance) with the pin documented in `safety.imageActBasis`; default is the frame-noun bank; emotion-terminal words are dead absolutely.

**R2 — Kind ↔ direction is a render-layer default, not a lookup (resolves C-13/X-03).** The KIND names the relation for the schema; the direction renders its SHAPE (biased balance→tension line; threshold→seam; retained motion→smear). Default map: Standoff/Figure-Ground→tension, Edge/Light-Fit→seam, Vector-into-Void→smear. Per-source override is legal WITH a written reason in the authoring note. SRC-02 is the canonical legal override: KIND=Figure-Ground but direction=seam, because its `field.balance` is authored "Centred" — a tension line would render near-flat and read as a false null, while the auger cut is the one line the figure drew in the field.

**R3 — The band is hand-authored relation-rank on the shared ladder, rendered as ONE `metTier` row (resolves the O2#1 / O5-rule-2 / O6-C-05 three-way).** O2 bans re-tiering a property Metrics already banded; O5 said derive from `event.charge`; O6 recommended a word with no meter chrome. Ruling: `auraField.tier` is a **hand-set word ranking the RELATION's staged tension** (matching `data.js`'s own "hand-authored, never computed" convention, `app.js:343-344`) — it may equal `event.charge` but never mechanically (S8's `src.event?.charge ||` fallback is vetoed; the renderer reads `auraField.tier` only). It renders as exactly one `metTier("Aura", tier, halo.a)` row — the meter-segments are the house's shipped expression of the word-ladder, so reusing them reads as house grammar, not a new instrument; doc 02's ban lands on NEW measurement visuals. Escape hatch: if visual QA reads the built plate as a gauge, drop the segments and keep the word (O6's fallback).

**R4 — Second-person ruled narrow (C-12, full rationale in the ledger).** Reading (b). One "you" clause survives in the worked set (SRC-01), deliberately, with a pre-authored strike-fallback.

**R5 — The pathetic-fallacy carve-out is REJECTED (tightens O3/S10).** O3 §5c invented an allowance ("'the lake holding its breath' passes because the freeze-line is literally crisp") that does not exist in the repo kill-list — `BR-AURA-RESEARCH.md:86` bans the pathetic fallacy flat, and the mission brief + O4 + O5 all kill that exact line. Ruling: strict kill; the safe move is a verb that names the optical fact ("the ice keeps the rest"). S10 row 1.6c must be corrected before the sheet is used.
