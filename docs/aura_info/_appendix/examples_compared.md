## KEY VERDICTS FOR DOSSIER BUILD

### 1. Class-Label Line (Aura top)
- **Format:** 1–3 words, two-word default, noun-ending fragment.
- **Vocabulary:** Prefer material/institutional/moral nouns over feeling-words. Examples: Quiet Labour, Warm Refusal, Cold Exit, Held Pressure.
- **Anti-pattern:** Avoid raw "energy," "vibe," "aura," "mood," "chaos," "magic," "soul," "essence" unless anchored to concrete noun.
- **Repo anchors:** SRC-01 = "Idle-Engine / Open-Palm / Northbound" (chips, not a class label yet). SRC-02 = "Auger-Braced / Sun-Struck / Low-Horizon" (chips).
- **Dossier opportunity:** Replace chips with one synthesized class line per source.

### 2. Aura Verdict Line (Aura body)
- **Syntax:** 2 beats (sometimes 3), 6–14 words, one contradiction, hardest word last.
- **Pattern A (default):** `[visible cue] reads as [charged contradiction]. [You …]` — transfers charge from person to image to viewer response.
- **Pattern B (premium):** Hide the hinge inside verbs; let `[image noun] + holds/keeps/lets/leaves + [charge]` work without repeating "reads as."
- **Voice:** Deadpan-archival > overclaim. Clinical precision over adjective piles. Concrete procedural nouns over abstractions.
- **Worked examples from research:** Doc 02 is the canon (SRC-01 "hand leaves first; warmth doesn't"; SRC-02 "cut is over, cold ringing stays"; SRC-03/04/05 set the polarity model).

### 3. Visual Treatment (Aura right)
- **Approved directions (doc 02):**
  1. **One-seam burn** — threshold, cut, window, break, puncture. Premium, highest non-redundancy.
  2. **Ghost smear** — proximity, passing motion, warmth, breath, touch retained. Very high premium feel.
  3. **Off-center tension line** — biased polarity (welcome/refuse, hold/release, shore/open). High premium, deeply descriptive of relations.
  4. **Type-burn** — only when phrase is singular enough. Riskiest (can steal Oracle's thunder).
- **Banned patterns:** Contour rings, vector overlays, heatmaps, rainbow gradients, chakra glows, sparkles, generic energy scanner UI. All read as second diagram or pseudo-mysticism.
- **Tint:** Restrained, derived from source Halo material. Keep in warm-silver archive palette. Inline SVG for crisp scaling and export.
- **Accessibility:** Left text explains visual. 3:1 graphics contrast, 4.5:1 text contrast. Semantic classes (`.dplate--aura`, `.daura__*`), never nth-of-type(). Prefers-reduced-motion honored.

### 4. Safety Enforcement (All Aura lines)
- **Gate 1 — Re-authorable cue:** Gesture, gaze, crop, light, styling, pose ONLY. Never body/face/build/health/identity.
- **Gate 2 — Grammatical subject:** Sentence is about the photograph (frame/gesture/light/setting), NOT the person as object to be evaluated.
- **Gate 3 — Connotation-strip:** Remove the photo-noun; does the predicate describe a STABLE DISPOSITION? If yes, fail. (e.g., "crouched brace" → "this person is humble" = fail.)
- **Swap test:** Would the sentence work with another sitter in the same pose, crop, light? If not, it was diagnosis.
- **Two-second test:** Could the viewer plausibly experience the effect within two seconds of looking? If it requires biography/trust/empathy, too far.
- **Four-zone filter:** Record (verifiable by archivist) / Read (argued from visible cues) / Reach (brief viewer consequence) / Ban (stable trait claims).

### 5. Non-redundancy Checks
- Aura must NOT repeat Diagram's left-panel labels verbatim.
- Aura must NOT repeat Surface Record's palette %, finish notes, material claims.
- Aura must NOT be the final mic-drop line; Oracle remains the only verdict.
- Aura is the last **felt** beat; Oracle is the last **spoken** beat.

### 6. Authored-First, Derived-Second
- **Never generate Aura copy directly.** Machine proposes anchor evidence (projector/container/rim/residue from source markers/analysis/diagram). Human writes the final sentence.
- **Schema:**
  - `id` (stable identifier)
  - `classLabel` (public class line)
  - `chips` (three short tags)
  - `reading` (one evocative sentence)
  - `anchors` (four AuraAnchor records: projector/container/rim/residue with cue + effect)

  - `visual.tint` (restrained tint from source halo)

  - `safety.noStableTraitClaim` (must be true)
  - `sourceRefs` (human-readable list of repo cues used)
  - `ariaSummary` (screen-reader text)
